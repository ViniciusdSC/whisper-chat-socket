# whisper-chat-socket

This is the chat service for whisper chat



### How to run:

- Create .env file and follow the style of .example.env file

- ```bash
  $ yarn install
  ```

- ```bash
  $ yarn build
  ```

- for development:

  - ```bash
    $ yarn dev
    ```

- for production:

  - ```bash
    $ yarn start
    ```
